
# О приложении

# Установка

`python3 -m venv venv`

`venv/scripts/activate`

`pip3 install -r requirements.txt`

# Использование

`python3 app/main.py`


# Генерирование документации

`python3 -m mkdocs build`

В папке **site** будет лежать статика

# Запуск документации

`python3 -m mkdocs serve`


# Тестирование

`python3 -m pytest test.py`
